Introduction

The GeneRepresentationPackage provides a set of S4 classes for protein-coding genes, long non-coding RNAs (lncRNAs), and microRNAs (miRNAs). These classes are inherited from a virtual Gene class, which includes common attributes such as ID, symbol, name, description, and genomic structure.

Creating Gene Objects

The package includes constructors for creating instances of ProteinCodingGene, lncRNAGene, and miRNAGene. These objects include specialized attributes for each gene type.

Creating a New Protein-Coding Gene Object

A ProteinCodingGene object inherits all attributes of the Gene class, along with proteinID and proteinSequence.

Example: Protein-Coding Gene

# Define a GRanges object for genomic structure
tp53_structure <- GRanges(
    seqnames = "chr17",
    ranges = IRanges(start = 7668402, end = 7687550),
    strand = "+"
)

# Create a ProteinCodingGene object
tp53 <- createProteinCodingGene(
    id = "ENSG000001",
    symbol = "TP53",
    name = "Tumor Protein",
    description = "Tumor suppressor gene",
    structure = tp53_structure,
    protein_id = "P04637",
    protein_sequence = "MVLSPADKTNVKAAWG"
)
print(tp53)

Accessor Methods

The package provides accessor methods to retrieve or modify attributes, such as symbol, structure, and name.

Getting the Symbol:

symbol(tp53)
#> [1] "TP53"

Setting the Symbol:

symbol(tp53) <- "TP53_updated"
symbol(tp53)
#> [1] "TP53_updated"

Accessing the Genomic Structure:

structure(tp53)
#> GRanges object with 1 range and 0 metadata columns:
#>       seqnames          ranges strand
#>          <Rle>       <IRanges>  <Rle>
#>   [1]    chr17 7668402-7687550      +
#>   -------
#>   seqinfo: 1 sequence from an unspecified genome; no seqlengths

Setting a New Genomic Structure:

new_structure <- GRanges(
    seqnames = "chr1",
    ranges = IRanges(start = 1000, end = 5000),
    strand = "-"
)
structure(tp53) <- new_structure
structure(tp53)
#> GRanges object with 1 range and 0 metadata columns:
#>       seqnames    ranges strand
#>          <Rle> <IRanges>  <Rle>
#>   [1]     chr1 1000-5000      -
#>   -------
#>   seqinfo: 1 sequence from an unspecified genome; no seqlengths

Computing Product Length

The lengthProduct method calculates the length of a gene’s product.

Example with a Protein-Coding Gene

lengthProduct(tp53)
#> [1] 16

Example with a lncRNA

lengthProduct(malat1)
#> [1] 11

Example with a microRNA

lengthProduct(mir21)
#> [1] 6
